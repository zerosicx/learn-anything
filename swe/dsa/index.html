Data Structures & Algorithms

A ground-up guide to DSA patterns in TypeScript, following the NeetCode roadmap

TypeScript 16 Chapters Interactive

This guide teaches data structures and algorithms from first principles using TypeScript. Each chapter builds on the last, covering the core patterns you need for technical interviews and real-world engineering. Code examples are runnable, and visualisations help you build intuition for how each structure behaves.

How to Use This Guide

Sequential or Jump Around

Chapters are ordered by dependency — earlier topics are prerequisites for later ones. Use the navigation bar at the top of each page to jump between chapters, or use the arrows to go sequentially. Each chapter includes TypeScript implementations, complexity analysis, and visual explanations.
